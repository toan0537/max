CACH CHAY WEB LOCAL SOLAIS
==========================

Thu muc project:
C:\Users\RYAN\.gemini\workspace\MAX\CO KHUE\CO KHUE\solais-local

Yeu cau:
- May da cai Node.js

CACH DE NHAT
------------
1. Chay file:
   CHAY-WEB-LOCAL.cmd

2. Mo web:
   http://127.0.0.1:3001

3. Khi khong dung nua, chay file:
   TAT-WEB-LOCAL.cmd

CACH CHAY THU CONG
------------------
1. Mo PowerShell hoac Command Prompt

2. Di vao thu muc project:
   cd /d "C:\Users\RYAN\.gemini\workspace\MAX\CO KHUE\CO KHUE\solais-local"

3. Chay server local:
   node .local-static-server.js . 3001 127.0.0.1

4. Mo web trong trinh duyet:
   http://127.0.0.1:3001

5. Khi khong dung nua thi tat server:
   Nhan Ctrl + C trong cua so terminal dang chay

NEU BI LOI EADDRINUSE
---------------------
Loi nay co nghia la cong 3001 dang co server khac dung roi.

Co 2 cach:
1. Neu day la web Solais da chay san, chi can mo:
   http://127.0.0.1:3001

2. Neu muon tat server cu roi chay lai:
   Chay file TAT-WEB-LOCAL.cmd
   Sau do chay lai CHAY-WEB-LOCAL.cmd

-------------
